Gambaran: HTML vs CSS vs JavaScript

HTML: Struktur/Kerangka

Menyusun konten: judul, paragraf, gambar, tautan, tabel, formulir.

CSS: Tampilan

Javascript: Perilaku

Menangani interaksi, logika, data dinamis, dan komunikasi jaringan.

Apa itu Browser?

Browser adalah aplikasi untuk render halaman web: membaca HTML, menerapkan CSS, menjalankan JavaScript, lalu menampilkan hasilnya.

Peran terhadap HTML

Membangun DOM Tree dari tag dan atribut.

Peran terhadap CSS

Menyusun CSSOM dan menghitung layout (render tree).

Peran terhadap JS

Menjalankan skrip untuk interaksi & manipulasi DOM.

HTML ‚Äî Dasar-Dasar

Apa itu HTML?

HyperText Markup Language ‚Äî bahasa markah untuk menyusun struktur konten web.

  • Tag: penanda seperti <h1>, <p>, <img>.
  • Elemen: tag pembuka + isi + tag penutup (kecuali elemen void seperti <img>).
  • Komentar: <!-- ini komentar -->
  • Browser juga mengenali markah lain: SVG, MathML (Selain HTML).

Struktur Utama (Anatomi HTML)

  • <!doctype html> ‚Äî memberi tahu browser mode standar.
  • <html lang="id"> ‚Äî akar dokumen & bahasa.
  • <head> ‚Äî metadata: judul, meta, link stylesheet, dsb.
  • <body> ‚Äî konten yang ditampilkan.

Elemen Semantik

Elemen semantik memberi arti pada struktur (membantu SEO, aksesibilitas, dan konsistensi).

<header> ‚Äî bagian atas halaman/section, biasanya logo & navigasi.

<nav> ‚Äî rak link navigasi.

<main> ‚Äî konten utama tunggal per halaman.

<section> ‚Äî kelompok topik/tema.

<article> ‚Äî konten mandiri (post, card berita).

<footer> ‚Äî informasi penutup/kaki halaman.

CSS ‚Äî Cascading Style Sheets

CSS mengatur presentasi dokumen: warna, ukuran, layout, tipografi, jarak, bayangan, animasi, responsif. Catatan: perilaku (logika) dikerjakan oleh JavaScript.

Struktur Aturan

selektor { property: value; }
.card { background: white; border-radius: 16px; padding: 24px; }

3 Cara Menambahkan CSS

  1. Inline: <h1 style="color: red">
  2. Internal: <style> ... </style> di <head>
  3. External: <link rel="stylesheet" href="style.css">

Selektor CSS

  • Tipe: p, img
  • Kelas: .card
  • ID: #hero
  • Relasi: nav a (descendant), ul > li (child)
  • Atribut: a[target="_blank"]
  • Pseudo-class: a:hover, input:focus
  • Pseudo-element: p::first-line

Properti CSS Populer

Teks & Warna

Box Model

width, height, margin, padding, border, border-radius, box-shadow

Layout

Flexbox ‚Äî Layout Fleksibel

  • Aktifkan: display: flex
  • Arah: flex-direction (row/column)
  • Sejajar utama: justify-content
  • Sejajar silang: align-items
  • Jarak: gap
  • Item: flex, order, align-self

Bootstrap vs Tailwind ‚Äî Cara Cepat Pakai CSS

Bootstrap

Framework komponen CSS (dengan JavaScript) yang menyediakan komponen siap pakai seperti navbar, kartu, modal, dan grid.

  • Pakai class yang sudah disediakan (mis. btn btn-primary).
  • Cocok untuk cepat jadi dengan gaya konsisten.

Tailwind CSS

Framework CSS utility-first ‚Äî fokus pada kelas utilitas kecil (mis. p-4, text-blue-500, flex) untuk menyusun desain fleksibel tanpa menulis CSS kustom panjang.

  • Kendalikan desain di markup dengan class utility.
  • Skala besar, konsisten, dan mudah dipakai untuk desain responsif.

Intinya:

Keduanya adalah framework CSS. Kita bisa membangun cepat dengan class name yang sudah disediakan ‚Äî pilih Bootstrap (komponen siap pakai) atau Tailwind (utility-first) sesuai selera & kebutuhan.
